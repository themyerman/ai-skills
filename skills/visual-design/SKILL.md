# visual-design

## What this is

Two concrete areas where shared vocabulary and technical specifics matter:

- **[print-production.md](print-production.md)** — the technical requirements for preparing files for print: resolution, color modes, bleed, trim, safe zones, file formats, and vendor prep. Has right and wrong answers.
- **[design-feedback.md](design-feedback.md)** — vocabulary for describing and critiquing visual work: composition, color relationships, visual weight, focal point, type hierarchy. Shared language makes feedback more useful than "something feels off."
