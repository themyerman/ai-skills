# digital-art-context

## What this is

A reference for informed conversation about digital art — where it came from, how it relates to photography, printmaking, and painting, and what the ongoing debates are. The goal is fluency: enough to situate work in context, engage with critics and collectors, and follow the live controversies without being encyclopedic.

This is not a production or tools skill. For print production (DPI, color profiles, bleed, file formats), see [visual-design / print-production](../visual-design/print-production.md).
